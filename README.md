## Project Structure

    `npm init -y`

### frontend
    `npx create-react-app app`
    `cd app`
    `npm install react-bootstrap bootstrap`
    `npm install react-router-dom`
    `npm install react-toastify`
    `npm install axios`
    `npm install`
    `npm start`



## Introduction to the project
### backend
    The backend part is in the api folder. It's entry is server.js. The db.js response for the interacting with mongoDB, 
    the routes.js response for the post, get, etc. operations.
    Most of the code is inheriate from assignment 8, only a few code added to routes.js (which is user login).

### front end
    The frontend part is in the app file.
    It is generated by creating React project.
    The assests folder is used to store the images that been used in the website.
    The component folder is used to store the components used in the React. The Util folder is used to store
    the components that been used in multiple pages.
